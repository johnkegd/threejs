## Project Setup

Download Node.js Run this followed commands:

1. Install dependencies (only the first time)

```bash
  npm install
```

2. Run the local server at localhost:8080

```bash
  npm run dev
```

3. (Optional) Build for production in the exercises/dist/ directory

```bash
  npm run build
```
